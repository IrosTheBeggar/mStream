# rust-server-audio Pre-built Binaries

Pre-built binaries for the mStream server audio player. These are built automatically by GitHub Actions.

Binary naming convention follows Node.js `process.platform` and `process.arch`:

- `rust-server-audio-win32-x64.exe` (Windows x64)
- `rust-server-audio-darwin-x64` (macOS Intel)
- `rust-server-audio-darwin-arm64` (macOS Apple Silicon)
- `rust-server-audio-linux-x64` (Linux x64)
